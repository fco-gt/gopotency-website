## Integración Directa

Implementar el middleware en tu enrutador es fácil.

```go
// Aplica el middleware a tu enrutador
router.Use(ginmw.Idempotency(manager))

// Tus manejadores existentes ahora están protegidos automáticamente
router.POST("/pagos", manejarPago)
```
